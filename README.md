> 세계 금융 뉴스를 관심사 키워드 기반으로 모아서 보여주는 실시간 뉴스 웹, 모바일 서비스 입니다. 

모야는 주식과 상품, 인덱스와 채권, 통화, 글로벌 이슈, 토픽, 섹터 등 전세계 32개국의 12,000개 매체로부터 관련 뉴스를 실시간으로 제공하는 서비스입니다.
